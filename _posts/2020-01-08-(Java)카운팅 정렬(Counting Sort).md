# 카운팅 정렬(Counting Sort)

### 요약

- 집합에 각 항목이 몇 개씩 있는지 세는 작업을 하여, 선형 시간에 정렬하는 알고리즘
- 정렬할 크기가 비교적 작을 때만 가능
- 제한 사항
  - `정수 또는 정수로 표현할 수 있는 자료`에 대해서만 적용 가능
  - 카운트들을 위한 충분한 공간을 할당하려면 `집합 내의 가장 큰 정수`를 알아야 한다.
- 시간 복잡도 : O(n + k) (n은 리스트 길이, k는 정수의 최대값)

### 코드

```
Counting_Sort(A, B, k)
// A [1 .. n] -- 입력 배열(1 to k)
// B [1 .. n] -- 정렬된 배열.
// C [1 .. k] -- 카운트 배열.

for i = 1 to k do
	C[i] = 0
for j = 1 to n do
	C[A[j]] = C[A[j]]+1
for i = 2 to k do
	C[i] = C[i] + C[i-1]
for j = n to 1 do
	B[C[A[j]]] = A[j]
	C[A[j]] = C[A[j]] - 1
```
