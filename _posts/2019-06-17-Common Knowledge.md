# Common Knowledge

### 인코딩 / 디코딩

- 대체, 치환, 전위

- 서로 다른 시스템간 소통을 위해 문자형식 변환
- 의미분자를 이스케이프 시킴
- 주요 정보 저장에 사용 X (∵방식이 오픈되어 있음)

### 시큐어 코딩 방법

1. 안전한 암호 알고리즘 사용 ⇒ 비도 보장
   - DES X ⇒ AES / MD X ⇒ SHA2↑
2. 적정한 키 길이 보장
   - 대칭키 : 128bit
   - 비대칭키 : 2048bit
3. 키 안전하게 보관
   - 물리적, 논리적 분리 - 접근 통제
